# Como executar o projeto

**Pré-requisitos:**

    Flutter >= 3.3.10

    Dart >= 2.18.6

**No terminal**
```bash
    # clonar repositório
    git clone https://github.com/LRittes/my_sesion.git

    # entrar na pasta do projeto
    cd my_sesion

    # fazer download dos packages necessários
    flutter pub get

    # rodar o projeto
    flutter run
```
